# login
## `POST` `/login`
### `application/x-www-form-urlencoded`  
|请求参数 | 类型 | 备注 |
|--------|-----|------|
username|必选|用户名|
password|必选|密码|

|返回参数|说明|
|------|---|
|date|返回消息|
|token|用户token|

|date|说明|
|---|---|
|"info":服务器错误|服务器错误|
|“密码错误”|`password`与`username`不匹配|
|”用户不存在“|`username`不存在|
|用户token|登录成功|
